# RoleBoard
Tablero multijugador para partidas de rol

## Prerequisitos

Instalar las dependencias
```
npm install --save
```

## Ejecución
Terminal 1: 
``` shell
$ cd RoleBoard
$ npm start
```

Terminal 2: 
``` shell
$ cd RoleBoard
$ npm run electron
```

Terminal 3: 
``` shell
$ cd RoleBoard/Server
$ node server.js
```
